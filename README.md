# React Todo App with Redux

## Overview

This project is a simple todo application built using React.js for the frontend and Redux for state management. It allows users to add, view, and manage their tasks effectively.

## Features

- **Navbar:** Navigation bar component for easy access to different sections of the app.
- **Create Todo:** Component for adding new tasks to the todo list.
- **Show All Todo:** Component to display all tasks added by the user.
